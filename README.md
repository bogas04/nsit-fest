# Nsit-fest
A react based frontend for NSIT fests.

# Build instructions
Make sure you have [babel cli](https://babeljs.io/docs/setup/#babel_cli)
```
npm i babel-cli -g
```
Then run the build script to transform ES2015 JSX files in src/js to dist/js
```
npm run build
```
