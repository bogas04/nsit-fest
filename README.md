# Nsit-fest
A react based frontend for NSIT fests.

### Build instructions
Clone the repository.
Make sure you have [npm](https://www.npmjs.com/) installed; then do
```
npm install
```
Then run the start script to setup an express server and get Main.js served
```
npm start
```
